# Mining-Simulator-Mod-Tools
## This is the official mod tools for Mining Simulator by JamesGames.
To install mods on windows press Window key + R then navigate to C:\Users\YourName\AppData\Roaming\Godot\app_userdata\mining Simulator\JamesGames\Mining Simulator\mods
and isert your pickaxe mod then load up the game and you should see your pickaxe in the shop!
